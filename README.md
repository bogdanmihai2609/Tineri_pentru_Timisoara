# 🌐 Site Web – Asociația „Tineri pentru Timișoara”

# DESCRIERE
Acest proiect este un site web informativ și interactiv pentru asociația **„Tineri pentru Timișoara”**, realizat cu scopul de a promova activitățile organizației și de a încuraja tinerii să se implice activ în comunitate. Website-ul permite prezentarea echipei, a proiectelor, precum și înregistrarea utilizatorilor printr-un sistem simplu de login/sign-up.

# FUNCTIONALITATE LOGIN/SIGN UP
- Autentificarea și înregistrarea sunt implementate folosind JavaScript și localStorage.
- Nu este necesar un server – totul este stocat și gestionat local, în browser.
- Parolele și utilizatorii se validează în mod simplu (doar ca exercițiu educativ, nu pentru producție).

# DESIGN SI INTERFATA
 Fundal cu imagine reprezentativă pentru orașul Timișoara.
- Font modern (Segoe UI)
- Elemente UI simple: formulare, butoane, secțiuni evidențiate.
- Interfața este intuitivă, gândită pentru utilizatori tineri.
